Exercício 10 - E tem alguma diferença?

Diga, em poucas palavras, qual a diferença entre os métodos auxiliares forEach e map.

O método forEach itera sobre cada elemento de uma lista, aplicando uma função para cada um deles.

Já o método map, itera sobre cada elemento de uma lista aplicando uma função e retorna um novo ARRAY com os elementos que passaram por aquela função (ele não altera o array original).
